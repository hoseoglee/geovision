# tool_stats — 도구/스킬 사용 통계

세션에서 사용한 도구/스킬 통계를 수집하고 보고하는 스킬.

## 워크플로우

1. `~/papyrus/records/` 내 작업 보고서(work_report) 파일들을 스캔한다.
2. 각 보고서에서 사용된 스킬/도구 정보를 추출한다.
   - frontmatter의 tags, 본문에서 스킬명 언급, `## 사용 도구` 섹션 등을 파싱
   - 보고서 파일명에서 날짜를 추출하여 기간별 분류
3. 기간별(일/주/월) 사용 통계를 생성한다.
   - 일별: 오늘 사용한 스킬/도구
   - 주별: 이번 주 사용 빈도 집계
   - 월별: 이번 달 사용 빈도 집계
4. 결과를 표 형식으로 사용자에게 보고한다.
5. 결과를 `~/papyrus/resources/도구_통계_{날짜}.md`에 저장한다.

## 출력 형식

```markdown
---
tags: [tool-stats, analytics]
date: {날짜}
---

# 도구/스킬 사용 통계 — {날짜}

## 요약
- 분석 기간: {시작일} ~ {종료일}
- 총 보고서 수: N
- 사용된 스킬 종류: M

## 스킬별 사용 빈도 (전체 기간)
| 순위 | 스킬 | 사용 횟수 | 비율 |
|------|------|-----------|------|
| 1    | work_report | 15 | 30% |
| 2    | research | 8 | 16% |
| ...  | ... | ... | ... |

## 주간 트렌드
| 주차 | 총 작업 수 | 주요 스킬 |
|------|-----------|-----------|
| W10  | 5 | work_report, research |
| ...  | ... | ... |

## 월간 트렌드
| 월 | 총 작업 수 | 주요 스킬 |
|----|-----------|-----------|
| 2026-03 | 12 | ... |
| ...     | ... | ... |

## 참고자료 (References)
없음
```

## 분석 대상
- `~/papyrus/records/` 하위의 모든 `.md` 파일
- 파일명 패턴: `YYYY-MM-DD-HH-MM_*.md` (work_report 스킬이 생성하는 형식)

## 추출 방법
- **frontmatter tags**: `tags` 필드에서 스킬명과 관련된 태그 추출
- **vessel ID**: 보고서 내 vessel 정보로 어떤 환경에서 작업했는지 분류
- **본문 키워드**: 스킬명이 본문에 직접 언급된 경우 카운트
- **날짜 기반 집계**: 파일명 또는 frontmatter의 date 필드로 기간별 분류
